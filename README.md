# AI Image Label Generator (AWS Rekognition)

This project is a web app that detects objects in images using AWS Rekognition.

## Features
- Upload image
- Detect objects using AI
- Draw bounding boxes
- Show confidence scores
- Download results as CSV

## Tech Stack
- Python
- Streamlit
- AWS S3
- AWS Rekognition

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Configure AWS:
aws configure

3. Run app:
streamlit run app.py
